# Security Policy

## Supported Versions

This project is currently maintained on the latest release.

## Reporting a Vulnerability

If you discover a security issue, please report it responsibly.

- Preferred: open a **private** security advisory on GitHub (Security → Advisories)
- Alternative: email the maintainers (if an email is published in the repo)

Please include:

- A description of the issue and potential impact
- Steps to reproduce (or a proof of concept)
- Any relevant logs/screenshots

We’ll confirm receipt and work on a fix as soon as possible.
